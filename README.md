# Modern React from the Beginning — Course exercises

Personal workspace for exercises from [**Modern React from the Beginning**](https://www.udemy.com/course/modern-react-from-the-beginning/) on Udemy, taught by [**Brad Traversy**](https://www.udemy.com/user/brad-traversy/). The exercises and structure follow that course. This repository is a **collection of small React apps**, each in its own folder with its own dependencies and scripts.

## Prerequisites

- [Node.js](https://nodejs.org/) 20+ (LTS recommended), **or** s
- [Bun](https://bun.sh/) — each app includes a `bun.lock` file

## Projects

| Folder | Description |
|--------|-------------|
| [`crypto-dash/`](crypto-dash/) | Crypto dashboard with Chart.js, `react-chartjs-2`, React Router |
| [`lifecycle-playground/`](lifecycle-playground/) | Component lifecycle experiments (Vite + React) |
| [`notes-app/`](notes-app/) | Notes UI with React, Vite, Tailwind CSS v4 |
| [`rating-ui/`](rating-ui/) | Rating UI (minimal React + Vite) |
| [`shopping-cart-ui/`](shopping-cart-ui/) | Shopping cart with context, `json-server` mock API, Tailwind CSS v4 |
| [`simple-timer/`](simple-timer/) | Timer app with React, Vite, Tailwind CSS v4 |
| [`cdn-setup/`](cdn-setup/) | Static HTML snippet for CDN / script-tag setup |
| [`lesson-resources/`](lesson-resources/) | Extra course files (if any) | d

## Tech stack (typical)

- React 19  
- Vite 6–8 (varies by project)  
- TypeScript (most apps)  
- Tailwind CSS v4 (where listed in `package.json`)  
- ESLint
